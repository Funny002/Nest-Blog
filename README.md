# Nest-Blog
> [English] (From Deepl)

这是基于 Nest 和 Vue3 实现的博客系统。

This is a blog system based on Nest and Vue3.

服务端将使用 Typeorm 实现 MySQL 数据库的实现，Ioredis 实现令牌存储，Passports 实现 JWT Token，以及其他库实现。

The server side will use Typeorm for the MySQL database implementation, Ioredis for the token store, Passports for the JWT Token, and other library implementations.

前端将使用 Element Plus 将会提供 UI组件，Pinia 实现数据存储，Animate 将实现部分动画，以及其他的库实现。

The front-end will use Element Plus which will provide the UI components, Pinia for data storage, Animate for some of the animations, and other library implementations.
